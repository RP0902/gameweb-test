# 山野驾车冒险

一个纯前端的程序化驾驶小游戏。玩家驾驶越野车在起伏的山坡公路上旅行，沿途会遇见森林、草地、湖泊与花田等自然风景。公路将持续自动生成，挑战是保持速度与平衡，看看自己能行驶多远。

## 体验方式

在项目根目录启动一个静态服务器，例如：

```bash
python -m http.server 8000
```

然后访问 <http://localhost:8000> 即可开始游戏。

## 操作说明

- `↑` 或 `W`：加速
- `↓` 或 `S`：刹车 / 倒车
- `←` 或 `A`、`→` 或 `D`：微调车身平衡
- `Space`：切换自动巡航

右侧面板会实时显示行驶距离、车速与巡航状态。
